10 things I didn't know about HTML

Mathias Bynens

10 things I didn't know about HTML - Mathias Bynens (video)

This video had a lot of surprising information. Sometimes HTML does not pick up human errors like spelling mistakes or when you forget to use a semi-colon. If your webpage works, then you shouldn't worry about if the HTML code is complicated. If the page is functioning the way you want it to, then that works.

3 Things I learned while watching this video

  • Some HTML elements in the coding can only be checked by humans and a computer that tries to validate certain things will not pick up on all of the one the human would pick up on
  • Functionality of a HTML trumps validation
  • The semi-colon is more necessary than we think

Mobile First

Luke Wroblewski

Luke Wroblewski | Mobile First | live at An Event Apart | (video)

It is amazing how much technology can change over the years. The speaker was even saying that just a few years back, the razor phone was popular and it uses similar but not as advanced technology as we have today. With condesnsed pixels and internet access, cell phones have turned into smart phones which are now becoming more popular than computers and laptops.

3 Things I learned while watching this video

  • Responsive web designs should minimalize the screen to have the most important and useful information and tools on it
  • Your web page should be easy to use on a mobile device especially if people will be using your website while they are on the go, such as a tracking app for flights or paying for a ticket online
  • The density of pixels on a mobile device can effect the use of a mobile site. If the pixels are not dense enough to show the small buttons or text on your mobile site, it won't be easily accessible
